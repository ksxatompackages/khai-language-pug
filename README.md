# Khai's Pug Grammars

A [Pug](http://pugjs.org) (formerly known as **Jade**) syntax highlighting for Atom,
derived from [Visual Studio Code](https://goo.gl/amSe0D)

## Installation

```bash
apm install khai-language-pug
```
